# One Day

_'Fix your day, don't try to fix your life'_

A personal habit builder I built for myself. Provides a useful framework and daily structure to keep you on track. Inspired by principles from the book 'Compound Effect' by Darren Hardy. Uses local storage for now. Persistent data for analytics.

### How to run:
npm install

npx expo export --platform web

npx serve dist 

Paste localhost:3000 into the browser
